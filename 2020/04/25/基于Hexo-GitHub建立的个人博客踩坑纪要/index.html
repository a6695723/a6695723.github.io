搭建前提准备：

  • node环境
  • GitHub账号
  • 安装有git，并配置了用户以及ssh密钥

大致流程为：

在GitHub上面初始化一个仓库，开放master主干为GitHub Pages的源文件，复制该仓库的ssh地址。在本地通过初始化hexo根目录，并安装所需依赖，配置_config.yml文件，安装hexo提供的上传插件，生成博客并上g传到GitHub上面
